# Daily Cal Update

Code and content related to updates for **The Daily Californian** website.

## Scope

This repository is used to:

- Draft and version HTML/CSS for the **Hall of Fame** page updates
- Draft and version HTML for **Board of Directors** bio pages
- Keep notes and questions for the Daily Cal tech/editorial teams

## Structure (planned)

- `hof/` — Hall of Fame related HTML snippets (landing page + inductee bios)
- `board/` — Board of Directors index page + individual bios
- `notes/` — Questions for Daily Cal, implementation notes, etc.

## Workflow

1. Build and test HTML locally (static files).
2. Once approved, copy the relevant HTML into the Daily Cal CMS (BLOX/TownNews).
3. Keep this repo as a source of truth for future tweaks.
